Community and development

Use GitHub Discussions for Q&A, gameplay feedback, Quest setup help, feature ideas, and community build reports. Use GitHub Issues for reproducible bugs or scoped development tasks.

Known current limitations

  • Blockiverse VR is in active development and is not a final store release.
  • Primary supported hardware is Meta Quest 3 and Meta Quest 3S with Quest Touch controllers.
  • Multiplayer is local LAN only. Cloud-hosted private worlds are tracked as future expansion work.
  • Worlds are bounded rather than infinite streaming terrain.
  • Voice communication uses Meta Quest party chat. There is no in-app voice chat.
  • Manual Quest hardware validation remains important for comfort, performance, audio timing, and release readiness.

Before reporting

  • Restart the headset and confirm the issue happens again.
  • Note whether you were in creative, survival, or LAN multiplayer.
  • For LAN multiplayer issues, include host/client device models and whether both devices were on the same local network.
  • For save/load issues, avoid deleting local app data until the save can be inspected or backed up.
